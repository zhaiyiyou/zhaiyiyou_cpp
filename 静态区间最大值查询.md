# 静态区间最大值查询

**题目名称**：静态区间最大值查询

**题目描述**： 给定一个长度为n的数组，和q个查询，每次查询区间\[l, r\]内的最大值。

**输入**： 第一行包含两个整数n和q（1 ≤ n ≤ 100000，1 ≤ q ≤ 100000），表示数组长度和查询次数。 第二行包含n个整数（1 ≤ 元素值 ≤ 1000000），表示数组元素。 接下来q行，每行包含两个整数l和r（1 ≤ l ≤ r ≤ n），表示查询的区间。

**输出**： 输出q行，每行包含一个整数，表示对应查询区间内的最大值。

**样例**：

    输入：
    5 3
    1 3 2 5 4
    1 3
    2 4
    3 5
    
    输出：
    3
    5
    5
